# Alura Challenge

## Projeto

Projeto da alura challenge. A ideia é criar uma aplicação web que permite
criar e compartilhar código em uma comunidade integrada!

## Tecnologias

O projeto foi desenvolvido usando React via (CRA -> https://reactjs.org/) e com Bootstrap 5 (https://getbootstrap.com/docs/5.0/getting-started/introduction/).

Foi usado para desenvolver a sintaxe a biblioteca https://highlightjs.org/

As rotas foram  criadas usando https://reactrouter.com/
